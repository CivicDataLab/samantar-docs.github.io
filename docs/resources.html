Resources¶

Machine Translation Datasets¶

Below are the Machine translation datasets for Indic Languages.

  • IIT-Bombay monolingual corpus, all back translated (45GB backtranslated data).

  • IIT Bombay English-Hindi Parallel Corpus: Largest en-hi parallel corpora in public domain. License - CC-BY-NC-4.0

  • PIB Multilingual Corpus: Sentence aligned parallel corpus between 11 Indian Languages, crawled and extracted from the press information bureau website.

  • Mann Ki Baat: The Prime Minister’s speeches on All India Radio, translated into many languages.

  • Charles University English-Tamil Parallel Corpus - License - CC-BY-NC-4.0

  • Charles University English-Odia Parallel Corpus - License - CC-BY-NC-4.0

  • Charles University English-Urdu Religious Parallel Corpus - License - CC-BY-NC-4.0

  • IndoWordnet Parallel Corpus: Parallel corpora mined from IndoWordNet gloss and/or examples for Indian-Indian language corpora (6.3 million segments, 18 languages). License - CC-BY-NC-4.0

  • MTurk Indian Parallel Corpus

  • CCMatrix

  • WikiMatrix Corpus

  • PMIndia: Parallel corpus from pmindia.org (research paper).

  • Indian Language Corpora Initiative: Available on TDIL portal on request.

  • OPUS corpus

  • WAT 2018 Parallel Corpus

  • Charles University English-Hindi Parallel Corpus: License - CC-BY-NC-4.0

  • TED Parallel Corpus

  • JW300 Corpus: Parallel corpus mined from jw.org.

  • ALT Parallel Corpus: 10k sentences for Bengali, Hindi in parallel with English and many East Asian languages.

  • FLORES dataset: English-Sinhala and English-Nepali corpora

  • Uka Tarsadia University Corpus: English-Gujarati sentence pairs

  • NLPC-UoM English-Tamil Corpus: 9k sentences, 24k glossary terms

  • English-Tamil Wiki Titles

  • JNU-BHLTR Bhojpuri Corpus: License - CC-BY-NC-4.0

  • EILMT Corpus

Machine Translation Pre-trained models¶

  • Multiway model trained on 7 languages, WAT 2019. Find the live demo here.

  • Multiway model trained on 11 languages, LREC 2020

Few Crawling code¶

  • jerinphilip/fairseq-ilmt: fairseq fork used to train multilingual models efficiently.

  • jerinphilip/ilmulti: Tokenizers, wrappers around fairseq for models trained on Indian languages.

  • shashanksiripragada/pib-crawl: Crawling code for Press Information Bureau website.

  • jerinphilip/mann-ki-baat: Crawling code for the Prime-minister’s speeches.

  • bhaddow/pmindia-crawler: Crawling code for pmdia.org
