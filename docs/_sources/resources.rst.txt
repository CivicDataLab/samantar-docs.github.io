Machine Translation Datasets
****************************

Below are the Machine translation datasets for Indic Languages.

- `IIT-Bombay monolingual corpus <https://drive.google.com/drive/folders/1p7_SMOq-DqOx86e22f1iC9srQw4QASr5?usp=sharing>`_, all back translated (45GB backtranslated data).
- `IIT Bombay English-Hindi Parallel Corpus <http://www.cfilt.iitb.ac.in/iitb_parallel/>`_: Largest en-hi parallel corpora in public domain. License - CC-BY-NC-4.0
- `PIB Multilingual Corpus <http://preon.iiit.ac.in/~jerin/resources/datasets/pib-v0.tar>`_: Sentence aligned parallel corpus between 11 Indian Languages, crawled and extracted from the press information bureau website.
- `Mann Ki Baat: The Prime Minister's speeches on All India Radio <http://preon.iiit.ac.in/~jerin/resources/datasets/mkb-v0.tar>`_, translated into many languages.
- `Charles University English-Tamil Parallel Corpus <http://ufal.mff.cuni.cz/~ramasamy/parallel/html/>`_ - License - CC-BY-NC-4.0
- `Charles University English-Odia Parallel Corpus <https://lindat.mff.cuni.cz/repository/xmlui/handle/11234/1-2879>`_ - License - CC-BY-NC-4.0
- `Charles University English-Urdu Religious Parallel Corpus <https://lindat.mff.cuni.cz/repository/xmlui/handle/11234/1-2582>`_ - License - CC-BY-NC-4.0
- `IndoWordnet Parallel Corpus <https://github.com/anoopkunchukuttan/indowordnet_parallel>`_: Parallel corpora mined from IndoWordNet gloss and/or examples for Indian-Indian language corpora (6.3 million segments, 18 languages). License - CC-BY-NC-4.0
- `MTurk Indian Parallel Corpus <https://github.com/joshua-decoder/indian-parallel-corpora>`_
- `CCMatrix <https://github.com/facebookresearch/LASER/tree/master/tasks/CCMatrix>`_
- `WikiMatrix Corpus <https://ai.facebook.com/blog/wikimatrix>`_
- `PMIndia <http://data.statmt.org/pmindia>`_: Parallel corpus from pmindia.org (`research paper <https://arxiv.org/abs/2001.09907>`_).
- `Indian Language Corpora Initiative <http://sanskrit.jnu.ac.in/ilci/index.jsp>`_: Available on TDIL portal on request.
- `OPUS corpus <http://opus.nlpl.eu/>`_
- `WAT 2018 Parallel Corpus <http://lotus.kuee.kyoto-u.ac.jp/WAT/indic-multilingual/index.html>`_
- `Charles University English-Hindi Parallel Corpus <https://lindat.mff.cuni.cz/repository/xmlui/handle/11858/00-097C-0000-0001-BD17-1>`_: License - CC-BY-NC-4.0
- `TED Parallel Corpus <https://github.com/ajinkyakulkarni14/TED-Multilingual-Parallel-Corpus>`_
- `JW300 Corpus <http://opus.nlpl.eu/JW300.php>`_: Parallel corpus mined from jw.org.
- `ALT Parallel Corpus <http://www2.nict.go.jp/astrec-att/member/mutiyama/ALT>`_: 10k sentences for Bengali, Hindi in parallel with English and many East Asian languages.
- `FLORES dataset <https://github.com/facebookresearch/flores>`_: English-Sinhala and English-Nepali corpora
- `Uka Tarsadia University Corpus <https://github.com/shahparth123/eng_guj_parallel_corpus>`_: English-Gujarati sentence pairs
- `NLPC-UoM English-Tamil Corpus <https://github.com/nlpc-uom/English-Tamil-Parallel-Corpus>`_: 9k sentences, 24k glossary terms
- `English-Tamil Wiki Titles <http://data.statmt.org/wikititles/v2/wikititles-v2.ta-en.tsv.gz>`_
- `JNU-BHLTR Bhojpuri Corpus <https://github.com/shashwatup9k/bho-resources>`_: License - CC-BY-NC-4.0
- `EILMT Corpus <http://tdil-dc.in/index.php?searchword=EILMT&searchphrase=all&option=com_search&lang=en>`_

Few Crawling code
*****************

- `jerinphilip/fairseq-ilmt <https://github.com/jerinphilip/fairseq-ilmt>`_: fairseq fork used to train multilingual models efficiently.
- `jerinphilip/ilmulti <https://github.com/jerinphilip/ilmulti>`_: Tokenizers, wrappers around fairseq for models trained on Indian languages.
- `shashanksiripragada/pib-crawl <https://github.com/shashanksiripragada/pib-crawl>`_: Crawling code for Press Information Bureau website.
- `jerinphilip/mann-ki-baat <https://github.com/jerinphilip/mann-ki-baat>`_: Crawling code for the Prime-minister's speeches.
- `bhaddow/pmindia-crawler <https://github.com/bhaddow/pmindia-crawler>`_: Crawling code for pmdia.org
