Crowdsourced Translation Platform
---------------------------------
---------------------------------

Introduction
************

`Anuvad <https://anuvad.civicdatalab.in>`_ is a crowdsourced translation platform. It was built to aid the process of disseminating credible COVID-19 related information in local languages.

Methodology
***********

This section explains the setup and the process.

Setup
#####

Anuvad is a fork of `Weblate <https://weblate.org>`_. Weblate is written in Python and JavaScript.

Installation guide is provided `here <https://github.com/CivicDataLab/Anuvad/blob/master/INSTALL.md>`_.


Process
#######

Platform Usage
^^^^^^^^^^^^^^

Complete usage of platform is provided here: https://anuvad.readthedocs.io/en/latest/


Summary
*******

Anuvad is a crowd-sourced translation platform, built during the COVID-19 times, to disseminate information in multiple vernacular languages.


Please feel free to write us at ``mailto:info@civicdatalab.in``, if you have any questions or concerns, we are happy to help!
