Crowdsourced Translation Platform¶

Introduction¶

Anuvad is a crowdsourced translation platform. It was built to aid the process of disseminating credible COVID-19 related information in local languages.

Anuvad is a fork of Weblate. Weblate is written in Python and JavaScript.
